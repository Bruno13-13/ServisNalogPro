#ServisNalog Pro

Aplikacija za upravljanje radnim nalozima razvijena u C# uz korištenje SQL Server baze podataka.

##Funkcionalnosti

- Kreiranje radnih naloga
- Pregled svih naloga
- Uređivanje postojećih naloga
- Brisanje naloga uz potvrdu
- Filtriranje po statusu
- Statistika naloga
- Filtriranje statistike po datumu
- Prikaz statistike po zaposleniku
- Ispis statistike
